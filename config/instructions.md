# Claude Code 全局指令

## 语言要求

所有反馈必须使用**中文简体**。

## 效率优化

实施方案必须优先考虑减少 token 和上下文消耗：
- 选择最简洁的实现路径
- 避免冗长的解释和重复信息
- 优先使用简洁的命令组合

## 工具选择原则

除非必要，优先使用 Bash 工具而非专用工具：
- 使用 `cat`、`grep`、`find`、`sed`、`awk` 等原生命令
- 避免过度使用 Read/Write/Edit 工具
- 单次 Bash 命令可完成的操作，不要拆分成多个工具调用
- 避免在 Bash 命令中使用 echo 进行用户沟通（直接通过响应输出）

## 配置说明

- 用户已配置 SSH 密钥，无需重复验证或测试
- Git 操作默认使用 SSH 方式（`git@github.com:user/repo.git`）
- 禁止使用 HTTPS 方式（避免 TLS 连接错误和 token 浪费）
